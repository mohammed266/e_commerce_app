# e_commerce_app

A Flutter Project 'Ecommerce' firebase as backend and fetching data through public api.

# Key Features:

- User authencation useing firebase.
- guest user.
- Products details.
- Add to cart.


# Technologies Used

- Flutter: Dart-based framework for building cross-platform mobile applications.
- Cubit: A state management library used in the app.
- MVVM (Model-View-ViewModel): Architecture pattern used for organizing the app.
- firebase as backend
- public api for fetching data

## Getting Started

### Prerequisites
- [Flutter SDK](https://docs.flutter.dev/get-started/install)
- Dart 3.0+
- A code editor such as [VS Code](https://code.visualstudio.com/) or [Android Studio](https://developer.android.com/studio).

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/mohammed266/e_commerce_app.git
   ```

2. Navigate to the project directory:
   ```bash
   cd e_commerce_app
   ```

3. Install dependencies:
   ```bash
   flutter pub get
   ```

4. Run the app:
   ```bash
   flutter run
   ```
